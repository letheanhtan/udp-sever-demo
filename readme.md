# UDP server demo

## Environment setup
- Python 3.9.9
- Sqlite 3.36.0

## Description
1. Client send message of temperature sensor data to server
2. Server read message from client and save sensor data into sqlite databse

## Demo step
1. Start udp server
`python udp_server.py`
2. Start udp client
`python udp_client.py`
